# Cross-Plan Glossary and Schema Dictionary

Provenance:
- `decision_id=DEC-0001`
- `change_request_id=RQ-028`

| Term | Canonical definition |
|---|---|
| evidence sufficiency | Minimum evidence mapping required for claims to proceed through release gates. |
| confidence label | Explicit trust level attached to outputs based on evidence coverage and caveat state. |
| production-safe | Eligible for external publish with all mandatory controls green. |
| blocked state | A non-progressing execution/publish state requiring remediation before continuation. |
| exception approval | Time-bounded, role-approved temporary override with audit trace and expiry. |
