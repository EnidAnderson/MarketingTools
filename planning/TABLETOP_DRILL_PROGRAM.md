# Tabletop Drill Program

Provenance:
- `decision_id=DEC-0001`
- `change_request_id=RQ-018`

## Cadence

1. Monthly tabletop drill.
2. Additional ad-hoc drill after any SEV-1 incident.

## Owners

1. Facilitator: Team Lead.
2. Security scenario owner: Security Steward.
3. Claims scenario owner: Product Steward.
4. Runtime/budget scenario owner: Platform Architect.

## Required scenarios

1. False claim shipped.
2. Cost runaway.
3. Role conflict deadlock.
4. Secret detection after commit.

## Success metrics

1. Containment steps identified within 15 minutes.
2. First 60-minute timeline completeness score >= 95%.
3. Action items have owners and due dates.

## Output requirements

1. Completed drill template.
2. Lessons-learned summary.
3. Control updates linked to policy docs and risk register.
