# INV-GLOBAL-GOV-002

## Invariant Statement
External publish requires two-person signoff (technical + business owner).

## Rationale
Reduces unilateral release risk.

## Scope
Any external-facing asset publication.

## Enforcement Points
`planning/EXTERNAL_PUBLISH_CONTROL.md`, release gate process.

## Evidence of Compliance
Publish record contains both required approver roles.

## Failure State
Single-person approval for external publish.

## Owner Role
Team Lead.

## Test Requirements
Approval validation fails without both role signatures.
