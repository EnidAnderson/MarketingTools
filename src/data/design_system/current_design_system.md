# Nature's Diet® Design System Specification

This document outlines the current design standards and specifications for the Nature's Diet® website. This information is extracted directly from the live website and CSS analysis to provide a precise and semantic understanding of our brand's visual identity.

## 1. Brand Identity

*   **Company Name:** Nature's Diet®
*   **Flagship Product:** Simply Raw®
*   **Logo Usage:**
    *   **Primary Logo:** The primary logo is a text-based design featuring "Nature's Diet®" in a custom font. The "O" in "Nature's" is replaced with a stylized paw print. It is prominently displayed in the website header.
    *   **Favicon:** A black square with the white initials "ND" and a white paw print above the "D".

## 3. Typography

Identify and list all font families, sizes, and weights used for different elements.

*   **Primary Font Family:** lulo-clean-w01-one-bold, oswald-medium
    *   **Usage:** Headings (H1, H2, etc.), Key Marketing Text
    *   **Sizes:**
        *   H1 ("Ready Raw"): 188px
        *   H2 ("Get Good Stuff"): 56px
        *   H3 ("IN STOCK & READY TO SHIP"): 25px
    *   **Weights:** Bold, Medium
*   **Body Text Font Family:** madefor-text-bold, helveticaneuew01-65medi, sans-serif
    *   **Usage:** Paragraphs, long-form content, input field labels

    *   **Weights:** Regular, Bold
*   **Call to Action (CTA) Font:** helvetica-w01-bold, "arial black"
    *   **Usage:** Buttons, important links
    *   **Sizes:** 16px
    *   **Weights:** Bold

## 4. Layout & Spacing

Describe the general layout principles, grid system (if any), and common spacing units.

*   **Page Width:** Max-width: 980px
*   **Grid System:** The layout is built on a responsive, component-based grid system rather than a traditional 12-column grid. Sections have a maximum width, and content within them is centered.
*   **Standard Spacing Units:** Spacing is primarily defined in pixels (px).
    *   Margin-bottom on list items: 12px
    *   Padding on buttons: Varies. e.g., Footer buttons have `padding: 5px` with additional horizontal padding for text.

## 5. UI Components

Describe standard styling for common UI elements.

*   **Buttons:**
    *   **Primary Button (Orange):**
        *   Background Color: Orange (#FC5C00)
        *   Text Color: White (#FFFFFF)
        *   Border Radius: 90px
        *   Hover State: Background becomes a darker orange (#A83D00), text remains white.
    *   **Secondary Button (White Outline):**
        *   Background Color: White (#FFFFFF)
        *   Text Color: Dark Gray (#323232)
        *   Border: 2px solid Dark Gray (#323232)
        *   Hover State: Background and border become Dark Gray (#323232), text becomes White (#FFFFFF).
*   **Forms (Input Fields):**
    *   Border: 1px solid Almost Black (#2F2F30)
    *   Background: White (#FFFFFF)
    *   Font: madefor-text-bold, 14px
*   **Navigation Links:**
    *   Default State: White text (#FFFFFF), no underline.
    *   Hover State: Background becomes Light Orange (#FDAD7F), text becomes Almost Black (#2F2F30).
    *   Selected State: Background becomes Primary Orange (#FC5C00), text becomes Almost Black (#2F2F30).

## 6. Imagery & Iconography

Describe the style and type of images and icons used.

*   **Image Style:** A mix of high-quality, professional product photography (showing packaging and ingredients) and lifestyle photography featuring healthy, active dogs and cats.
*   **Color Tone of Images:** Bright, warm, and natural, emphasizing the "natural" and "healthy" aspects of the brand.
*   **Iconography Style:** Simple, flat, single-color line icons and filled shapes (e.g., social media icons, cart icon, navigation arrows).
