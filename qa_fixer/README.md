# qa_fixer Home

This folder is the implementer home for incoming Team Lead hardening/testing requests.

## Invariant naming scheme

1. Invariant ID: `INV-GLOBAL-<DOMAIN>-<NNN>`
2. Request ID: `RQ-INV-<NNN>`
3. Test file: `scripts/tests/invariants/test_inv_global_<domain>_<nnn>.sh`

## Current focus

1. Maintain invariant test suites and machine-readable reporting.
2. Land additive source/process hardening fixes discovered by tests.
